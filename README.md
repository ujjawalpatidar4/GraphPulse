# GraphPulse - Workflow Builder

A visual workflow builder frontend application built with React.

## Features
- Create, Edit, and Delete workflow nodes.
- Support for Action (sequential) and Branch (conditional) logic.
- Visual tree representation with custom connectors.
- Interactive UI with inline editing and context menus.

## Technologies
- React (Vite)
- JavaScript
- CSS (Custom styling, no external UI libraries)

## How to Run

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
